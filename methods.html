Methods

What to build the analysis

1. Project Overview

This GIS project analyzed air quality in the Czech Republic using satellite-based data and geospatial techniques. The goal was to understand spatial and temporal patterns of two pollutants: NO₂ and PM2.5. We integrated pollution data with land cover and population data to explore links between exposure and land use.

2. Workflow and Methodology

  1. Data Acquisition: We used CAMS reanalysis data for NO₂ and PM2.5, ESA CCI Land Cover (2022), and WorldPop population data (2020).
  2. Preprocessing: Data were clipped to the Czech Republic, land cover was reclassified, and all layers were harmonized to the same projection.
  3. Analysis: We performed time-series statistics from 2013 to 2022, examined land use correlations, and calculated population exposure based on WHO thresholds.
  4. Visualization: Outputs included bivariate maps, annual average maps, and interactive WebGIS tools.

3. Tools Used

  • QGIS with GRASS and SAGA processing toolboxes
  • GeoServer and OpenLayers for WebGIS deployment
  • Data sourced from the Copernicus Climate Data Store (CDS)

4. Key Outputs

  • Annual NO₂ and PM2.5 maps (2013–2022)
  • Land cover correlation analysis
  • Population exposure estimations
  • Bivariate maps of pollution and population
  • Interactive WebGIS for public access
