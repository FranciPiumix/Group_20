1. Introduction

This section describes the methodology adopted in our GIS project focused on CZ Pollution monitoring. It includes data collection, preprocessing and spatial analysis techniques.

2. Data Used

  • Satellite imagery from Sentinel-2 (Copernicus Programme)
  • Air quality monitoring data from ARPA
  • OpenStreetMap for roads and land use
  • Digital Elevation Model (DEM)

3. Processing Steps

  1. Download and preprocess satellite data (atmospheric correction, clipping)
  2. Convert raster data to NDVI for vegetation analysis
  3. Overlay pollution data with land use layers
  4. Identify high-risk zones using spatial interpolation
  5. Generate maps and summary statistics
